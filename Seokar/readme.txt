== 📥 Installation Guide / راهنمای نصب ==

🔹 **Method 1: Automatic Installation via WordPress**  
🔹 **روش ۱: نصب خودکار از وردپرس**  
1. Log in to your **WordPress Admin Dashboard**.  
1. وارد **پیشخوان وردپرس** شوید.  
2. Navigate to **Appearance → Themes → Add New**.  
2. به مسیر **نمایش → پوسته‌ها → افزودن جدید** بروید.  
3. Search for "Seokar" and click **Install**.  
3. "Seokar" را جستجو کنید و روی **نصب** کلیک کنید.  
4. After installation, click **Activate**.  
4. بعد از نصب، روی **فعال‌سازی** کلیک کنید.  

🔹 **Method 2: Manual Installation via Host**  
🔹 **روش ۲: نصب دستی از هاست**  
1. Download the **Seokar.zip** file from the official website.  
1. فایل **Seokar.zip** را از سایت رسمی دانلود کنید.  
2. Log in to your **Hosting Control Panel (cPanel or DirectAdmin)**.  
2. وارد **مدیریت هاست (cPanel یا DirectAdmin)** شوید.  
3. Upload and extract the file to the `/wp-content/themes/` directory.  
3. فایل را در مسیر `/wp-content/themes/` آپلود و استخراج کنید.  
4. Activate the theme from **WordPress Admin Dashboard → Appearance → Themes**.  
4. از **پیشخوان وردپرس → نمایش → پوسته‌ها** قالب را فعال کنید.

== 🎨 Customization and Settings / تنظیمات و سفارشی‌سازی ==
- Go to **WordPress Admin Dashboard → Appearance → Customize**.  
- به **پیشخوان وردپرس → نمایش → سفارشی‌سازی** بروید.  
- Modify **Header, Footer, Typography, Color Scheme, and SEO** settings without the need for additional plugins.  
- تنظیمات **هدر، فوتر، تایپوگرافی، رنگ‌بندی و سئو** را بدون نیاز به افزونه تغییر دهید.  
- If using **Elementor**, enable the theme's custom elements.  
- در صورت استفاده از **Elementor**، المان‌های اختصاصی قالب را فعال کنید.
